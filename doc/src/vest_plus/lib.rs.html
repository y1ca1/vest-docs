1//! # VEST++: The next generation VErified Serialization Toolkit
2//!
3//! Vest++ provides a library of formally verified (de-)serializer **combinators** built with
4//! [Verus](https://github.com/verus-lang/verus), as well as a domain-specific language (DSL) for defining
5//! complex binary data formats leveraging these formally verified combinators.
6//!
7//! ## Key Properties
8//!
9//! All Vest++ combinators are formally proven to satisfy **[Serialize-Parse Roundtrip](core::proof::SPRoundTrip)**:
10//! for any value `v` consistent with a format, serializing `v` and then parsing the result recovers `v`.
11//! Additionally, combinators that are proven to be **[Non-Malleable](core::proof::NonMalleable)** also satisfy
12//! **[Parse-Serialize Roundtrip](core::proof::PSRoundTrip)**: parsing a buffer and then serializing the result
13//! reproduces the consumed prefix of the original buffer.
14//!
15//! ## A Taste of the DSL and Combinators
16//!
17//! A simple TLV (tag-length-value) message can be expressed as
18//!
19//! ```vest
20//! tlv = {
21//!   @t: u8,
22//!   @l: u16,
23//!   v: [u8; @l] >>= choose(@t) {
24//!     0x01 => Vec<msg1>,
25//!     0x02 => msg2,
26//!     0x03 => msg3,
27//!     _ => void, // reject other tag values
28//!   }
29//! };
30//! ```
31//!
32//! The same format can be expressed using combinators as
33//!
34//! ```rust
35//! let tlv =
36//!     Bind(
37//!     (U8, U16Le),
38//!     TLVOf(
39//!       TVOr(0x01, RepeatUntilEof(Msg1),
40//!       TVOr(0x02, Msg2,
41//!       TVOr(0x03, Msg3,
42//!       Uninhabited()))))
43//! );
44//! ```
45//!
46//! See the [`combinators`] module for the full catalog of available combinators.
