Expand description

DPS-specific serializability constraints.

Some combinators impose conditions on which (value, obuf) pairs are valid for DPS serialization. For example, crate::combinators::Opt requires the inner parser to fail on obuf when the value is None; crate::combinators::Star requires the inner parser to fail on obuf for termination; crate::combinators::Choice requires the left parser to fail when serializing the right branch.

Most primitive combinators leave this as true.
