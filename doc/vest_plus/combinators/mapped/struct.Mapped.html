Struct Mapped 

Source 
pub struct Mapped<Inner, M> {
    pub inner: Inner,
    pub mapper: M,
}
Expand description

Isomorphic format transformation combinator.

Mapped { inner, mapper } applies a bijective mapping to transform the inner combinator’s value type. The binary format is identical to inner’s.

Consistency, byte length, unambiguity, and non-malleability are all inherited from inner (through the bijection).

Fields§

§inner: Inner

The inner combinator whose values are being transformed.

§mapper: M

The bijective mapping between the inner and outer value types.
